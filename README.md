# MayaFlix--Movie-Recommender
🎬 MayaFlix is an AI-powered offline Movie Recommendation System built using Python, Flask, Pandas &amp; Scikit-learn 🤖. It recommends similar movies using Content-Based Filtering and Cosine Similarity 🎥✨ with a modern Netflix-inspired dark UI 🌙, responsive design 📱, and fully local dataset support without APIs 🚀.
